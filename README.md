# APEX-FITNESS---Elite-Performance-Training-Website
 
A high-end, conversion-focused gym website designed to deliver a **modern fitness brand experience** with dynamic scheduling, fitness calculators, and engaging UI interactions.

This project simulates a **premium fitness brand website** built with pure frontend technologies and optimized for **user engagement, mobile usability, and high conversion potential**.

---

# 🚀 Live Experience

The website includes interactive components such as:

• Dynamic class schedule system  
• BMI calculator  
• Daily calorie requirement calculator  
• FAQ accordion  
• Smooth scroll navigation  
• Sticky call-to-action buttons  
• Parallax hero effects  
• Contact form UI  
• Mobile navigation menu  

All features are built using **vanilla JavaScript**, ensuring lightweight performance and fast loading.

# 🧩 Key Features

## 🏋️ Premium Fitness Landing Page
A modern hero section with parallax scrolling and brand-focused typography.

## 📅 Dynamic Class Schedule
Classes are rendered dynamically using JavaScript objects.

Features include:

- Daily class switching
- Coach display
- Duration information
- Available seats tracker
- Booking confirmation simulation

## 🧮 Fitness Calculators

### BMI Calculator
Users can calculate their Body Mass Index and instantly see their category:

- Underweight
- Normal
- Overweight
- Obese

### Calorie Calculator
Daily calorie requirements are calculated using the **Mifflin-St Jeor Equation** with activity multipliers.

The result also suggests:

- Calories for fat loss
- Calories for muscle gain

## ❓ FAQ System
Expandable FAQ accordion for common gym questions.

## 📩 Contact / Free Trial Form
A conversion-focused form allowing users to claim a **free 7-day trial**.

Includes UI submission feedback simulation.

## 📱 Fully Responsive Layout
Optimized for mobile devices with a collapsible navigation menu.

## 🎨 Smooth Animations
Uses **Intersection Observer API** to trigger animations as elements enter the viewport.

## 📌 Sticky Call-To-Action
A persistent CTA bar encourages users to join the gym or start a trial.

## 💬 WhatsApp Chat Button
Provides instant messaging access for quick inquiries.

# ⚙️ How It Works

### Schedule System
A JavaScript object stores weekly class data.

When a user selects a day, the schedule dynamically renders the available classes.

### Booking System
Users can click **Book**, which triggers a confirmation dialog and simulates a booking success message.

### Fitness Calculators
The calculators take user inputs and compute results instantly using JavaScript formulas.

### Animation System
Elements marked with `.fade-up` are observed using the **Intersection Observer API** and animated when visible.
